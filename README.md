# acontext-skills

Agent skills for building AI agents with [Acontext](https://acontext.io) SDK integration - persistent storage, file operations, and Python sandbox execution.

## Install

```bash
npx skills add mbt1909432/acontext-skills --skill acontext-agent-integration
```

## Quickstart

After installation, try these prompts with your coding agent:

**Build a chatbot:**
```
Build a basic chatbot with Acontext that stores conversation history
```

**Add file tools:**
```
Add disk tools to my agent so it can read and write files
```

**Python execution:**
```
Set up a Python sandbox for data analysis with chart generation
```

**Full agent:**
```
Build an AI agent with Acontext that has conversation memory, file storage, and Python code execution
```
